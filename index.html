关于Java参数是值传递还是址传递的问题研究

发表于 2020-06-16 | 分类于 Java

结论

Java中所有参数都是值传递。
既然是值传递，那么它就是一份拷贝（址传递才是原件），拷贝件的改变不会影响原件。
